# react-antd-router-sass-css-loader
react+antd+路由+sass+css-loader 脚手架
# 如何使用
##命令行安装
```shell
git clone https://github.com/suings/react-antd-router-sass-css-loader.git project-name
cd project-name
yarn
yarn start
```
or
```shell
git clone https://github.com/suings/react-antd-router-sass-css-loader.git project-name
cd project-name
npm install
npm start
```
or
```shell
git clone https://github.com/suings/react-antd-router-sass-css-loader.git project-name
cd project-name
cnpm
cnpm start
```
